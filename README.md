# 声明

一个基于spring cloud编写的博彩系统，基本的功能已实现(购彩大厅，投注，追号，充值，提现，代理返点等)。
源码只供学习使用，请勿将系统用于非法业务，如用于商业活动与本人无关，欢迎大家一起来维护提交自己的push！
